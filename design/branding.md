# 🎨 Manual de Branding Final: Grid

## 🤖 Identidad Visual
- **Nombre de la App:** Grid
- **Eslogan:** Crea, Organiza, Avanza.
- **Icono Principal:** Cabeza de robot azul redondeada con dos antenas. Cara de pantalla negra con un prompt de terminal en cian: `>_`.
- **Persona del Asistente:** Orión (omnipresente vía burbuja flotante).

## 📝 Lenguaje de Diseño
- **Estilo:** Dark Mode Profesional (SaaS High-End).
- **Colores:** Discord Blurple (#5865F2), Deep Charcoal (#313338), Púrpura Profundo (#7289DA).
- **Tipografía:** Sans-serif moderna (Inter/Roboto). Limpia y legible.

## 💬 Tono de Comunicación
- Directo, minimalista y funcional. Sin relleno.
