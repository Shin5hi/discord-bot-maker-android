# 🤖 Resumen del Proyecto: Grid - Discord Bot Maker

Este documento centraliza toda la visión del proyecto para que el equipo de ingeniería (Codex, Sprite, Gaul) esté alineado.

## 📌 Visión General
Crear la primera aplicación nativa para Android que permita a los usuarios construir, configurar y lanzar bots de Discord directamente desde su móvil sin tocar código.

## 👥 Equipo del Studio
- **Gaul (Yo):** Arquitecto de Sistemas y líder de UI/UX.
- **Donna:** Diseñadora gráfica (Estilo Discord Oficial).
- **Codex:** Orquestador de programación y lógica de Backend.
- **Sprite:** Especialista en lógica de IA y personalidad del asistente.

## 📟 Componentes Clave
1. **Server Tree View (El Árbol):** Biblioteca de herramientas organizada por categorías colapsables (Moderación, Economía, etc.) con alta densidad y líneas de jerarquía.
2. **Asistente Orión:** Burbuja flotante global con el logo del bot azul. Ofrece ayuda minimalista y concisa.
3. **Integración Discord:** Sistema de login OAuth y gestión de tokens segura.
