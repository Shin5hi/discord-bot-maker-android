# Discord Bot Maker - Public Brand Kit

This kit documents the public-facing visual system for Discord Bot Maker. It is a local, editable asset set for Shin5hi's personal project presentation work.

## Purpose

Discord Bot Maker should feel like a friendly Android-first control room for running a Discord bot from a phone. The visual identity combines a rounded bot shell, a terminal-inspired `>_` face, Discord blurple, cyan runtime energy, and dark technical surfaces.

## Final Assets

| Asset | Use |
| --- | --- |
| `profile-mark.svg` / `profile-mark.png` | Main public face, social preview, hero lockup. |
| `logo-lockup.svg` | Compact README or documentation lockup. |
| `readme-preview.svg` | Lightweight README visual preview. |
| `app-icon-branding.svg` / `app-icon-branding.png` | Launcher icon mask study. |
| `brand-presentation.svg` / `brand-presentation.png` | Single-slide public overview. |

## Public vs Personal Use

- Public presentation: use `brand-presentation` or the `public-brand-deck` slides.
- Personal reference: use `personal-brand-kit` when reviewing the full system at a glance.
- README/GitHub: use `logo-lockup`, `readme-preview`, and optionally `brand-presentation`.
- Future store prep: use the icon system as a base, but create store-specific assets separately.

## Typography

The current SVG assets use `Segoe UI` as a reliable Windows-rendered fallback. The direction is:

- Heavy sans for identity and slide titles.
- Regular sans for supporting copy.
- Monospace only for terminal cues such as `>_`.

## Usage Rules

- Keep the approved logo geometry unchanged.
- Scale the bot, wordmark, glow and Android badge as one visual unit.
- Use dark backgrounds for hero and profile surfaces.
- Keep enough air around the antennas and terminal face.
- Use white masks only when demonstrating launcher icon behavior.
- Avoid recoloring the mark outside blurple, cyan, ink, white and Android green.
- Do not place the mark inside visually busy compositions.
- Do not create tiny duplicate lockups where the glow, badge or terminal face become unreadable.
